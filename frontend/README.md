## Getting Started

1.  Make sure you are on the `Front-end` branch.
2.  Navigate to the `frontend` directory:
    ```bash
    cd frontend
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
    *(Note: Ensure your Node.js and npm are installed correctly. If `npm` fails, reinstall Node.js)*
4.  Start the app:
    ```bash
    npx expo start
    ```
5.  Scan the QR code with your phone (Expo Go app) or run on emulator.
