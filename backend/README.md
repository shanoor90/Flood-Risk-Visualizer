## Getting Started

1.  Make sure you are on the `Back-end` branch.
2.  Navigate to the `backend` directory:
    ```bash
    cd backend
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
    *(Note: Ensure your Node.js and npm are installed correctly. If `npm` fails, reinstall Node.js)*
4.  Start the server:
    ```bash
    npm run dev
    ```
    (Or using node directly: `node server.js`)

## API Endpoints
- `GET /` - Health check
- `GET /api/status` - Example status endpoint
